# Snake Game

An implementation of the classic game in a modern JS framework.

## Setup

```bash
$ git clone git@github.com:JulianHinsch/snake.git
$ cd client
$ npm install && npm build
$ cd ..
$ npm install && npm run start
```

## Run Tests

```bash
$ cd client && npm test
```

## Transpile SCSS

```bash
$ cd client && npm run sass
```
